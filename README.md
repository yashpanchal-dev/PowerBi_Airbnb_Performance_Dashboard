# PowerBi_Airbnb_Performance_Dashboard

This project presents an interactive **Airbnb Listings Analysis Dashboard** built using **Power BI** to analyze listing prices, availability, reviews, and neighborhood performance. The dashboard helps understand market trends, customer preferences, and pricing patterns across different locations.

---

## 📌 Project Overview

The objective of this project is to analyze Airbnb listing data and generate insights that can help:

- identify high-performing neighborhoods  
- understand pricing trends  
- analyze host activity  
- evaluate customer engagement using reviews  
- monitor listing availability  

The dashboard transforms raw Airbnb dataset into meaningful visual insights for better business and customer decision-making.

---

## 🛠 Tools & Technologies Used

- Power BI Desktop  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Data Visualization Techniques  

---

## 📊 Key Analysis Performed

### ✅ 1. Price Analysis
- Analyzed average price of listings
- Compared prices across neighborhoods
- Identified high-value and low-value areas

### ✅ 2. Listings Distribution Analysis
- Total number of listings by neighborhood
- Most popular areas with maximum listings
- Comparison of room types

### ✅ 3. Availability Analysis
- Analyzed availability of listings throughout the year
- Identified areas with high and low availability

### ✅ 4. Review & Customer Engagement Analysis
- Total number of reviews
- Most reviewed listings and neighborhoods
- Customer activity insights

### ✅ 5. Host Performance Analysis
- Identified hosts with multiple listings
- Analyzed contribution of hosts to the overall market

---

## 📈 Dashboard Features

- KPI Cards (Total Listings, Average Price, Total Reviews)
- Interactive Filters / Slicers
- Bar Charts and Column Charts
- Location-based analysis
- Dynamic and interactive visualizations
